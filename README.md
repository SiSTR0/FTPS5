# FTPS5
FTP server for PS5 Webkit

# Building

Needs gcc 9 installed.

# Custom commands:

- `MTRW`: Remount `/system` and `/system_ex` partitions with read-write access

## Known issue

Forking webkit process causes a kernel panic when the console is shut down.
